# Stocktwits-challenge

## About

Application allows user to input a US stock symbol or several symbols (eg: AAPL, BABA, BAC, etcâ€¦) and pull tweets from StockTwits that mention this symbol, then displays the latest matching tweets.
When a new tweet comes in, it automatically added, without having to refresh a page.
The user is able to later add/remove symbols as well. If a user deletes a symbol, the matching tweets also disappear.
The page is responsive.

## Technologies used
* react.js
* express.js
* stocktwits api
